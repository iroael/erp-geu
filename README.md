## Quick Start

```bash [Terminal]
pnpm i # If you don't have pnpm installed, run: npm install -g pnpm
```

## Contributing

1. Clone this repository.
2. Install dependencies `pnpm install`.
3. Use `pnpm run dev` to start dev server.
